## Qt 绘图与双缓冲机制实战笔记

在 Qt 图形开发中，掌握窗体位置函数、QPainter 绘图框架和双缓冲机制是实现流畅绘图的核心。这篇笔记就来梳理这些关键知识点，帮你快速上手 Qt 绘图开发。

## 一、Qt 窗口位置相关 API 函数

Qt 提供了一系列获取窗体位置和显示区域大小的函数，这些函数是绘图定位的基础。<img src="D:\DownLoad\56cfca13f3c34aeba11b5fe79df527b5.pdf_0_2400.jpg~tplv-a9rns2rl98-resize-crop_252_656_1178_1358_926_702.jpeg" alt="56cfca13f3c34aeba11b5fe79df527b5.pdf_0_2400.jpg~tplv-a9rns2rl98-resize-crop_252_656_1178_1358_926_702" style="zoom:50%;" />

1. **核心函数及作用**

   - `x()`/`y()`：获取窗体左上角的坐标
   - `width()`/`height()`：获取窗体的宽度和高度
   - `rect()`：返回窗体客户区的矩形区域，坐标以客户区左上角为原点
   - `geometry()`：返回窗体在屏幕上的几何信息，包含位置和大小
   - `frameGeometry()`：返回包含窗口边框的整个窗口的几何信息

2. **测试结果参考**

   实际测试中，这些函数会返回具体数值，比如：

   - `geometry()`: 1182, 339, 300, 150
   - `width()`: 300  `height()`: 150
   - `rect()`: 0,0,300,150

   ```cpp
   // 核心函数：更新所有标签的显示值，获取当前窗口的位置/大小API结果
   void Dialog::UpdateLabelFunc()
   {
       // ========== 1. 获取geometry()值并显示 ==========
       // geometry()：返回窗口的几何信息（相对于父窗口/屏幕的x,y坐标 + 宽度、高度）
       // x()=窗口左上角横坐标，y()=窗口左上角纵坐标，width()=宽度，height()=高度
       QString strgeometry;  // 存储geometry()的拼接结果
       QString str1,str2,str3,str4; // 临时存储各数值的字符串形式
   
       // 将数值转为字符串并拼接："x,y,width,height"
       strgeometry = str1.setNum(geometry().x()) + "," + 
                     str2.setNum(geometry().y()) + "," +
                     str3.setNum(geometry().width()) + "," + 
                     str4.setNum(geometry().height());
       labelgeometryvalue->setText(strgeometry); // 显示到对应标签
   
       // ========== 2. 获取width()/height()值并显示 ==========
       // width()：返回窗口客户区的宽度（不含边框/标题栏）
       // height()：返回窗口客户区的高度（不含边框/标题栏）
       QString strw,strh;
       labelwidthvalue->setText(strw.setNum(width()));    // 显示宽度值
       labelheightvalue->setText(strh.setNum(height()));  // 显示高度值
   
       // ========== 3. 获取rect()值并显示 ==========
       // rect()：返回窗口客户区的矩形区域，x()和y()恒为0（相对于窗口自身），仅包含宽高
       // 注意：这里代码中用了width()/height()替代rect().width()/rect().height()，效果一致
       QString strrect;
       QString strrect1,strrect2,strrect3,strrect4;
       strrect = strrect1.setNum(rect().x()) + "," + 
                 strrect2.setNum(rect().y()) + "," +
                 strrect3.setNum(width()) + "," + 
                 strrect4.setNum(height());
       labelrectvalue->setText(strrect); // 显示到对应标签
   
       // ========== 4. 获取size()值并显示 ==========
       // size()：返回窗口的尺寸（仅包含宽高，无坐标信息）
       QString strsize;
       QString strsize1,strsize2;
       strsize = strsize1.setNum(size().width()) + "," + 
                 strsize2.setNum(size().height());
       labelsizevalue->setText(strsize); // 显示到对应标签
   }
   
   // 重写窗口大小变化事件：当窗口被拉伸/缩小，自动触发此函数
   // 参数：QResizeEvent* 包含大小变化的信息
   void Dialog::resizeEvent(QResizeEvent *)
   {
       // 窗口大小变化时，重新更新所有标签值，实现实时显示
       UpdateLabelFunc();
   }
   
   // 重写窗口移动事件：当窗口被拖动位置，自动触发此函数
   // 参数：QMoveEvent* 包含位置变化的信息
   void Dialog::moveEvent(QMoveEvent *)
   {
       // 窗口位置变化时，重新更新所有标签值，实现实时显示
       UpdateLabelFunc();
   }
   ```

**代码的核心API**

|         函数         |        作用        |                     关键特点                     |
| :------------------: | :----------------: | :----------------------------------------------: |
|     `geometry()`     |  获取窗口几何信息  | 包含屏幕坐标 (x,y) + 宽高，受窗口移动 / 缩放影响 |
| `width()`/`height()` |   获取客户区宽高   |          仅宽高，无坐标，受窗口缩放影响          |
|       `rect()`       | 获取客户区矩形区域 |        x/y 恒为 0（相对窗口自身），仅宽高        |
|       `size()`       |    获取窗口尺寸    |  仅宽高，无坐标，和 width ()/height () 结果一致  |

**事件重写机制**

- `resizeEvent`：窗口大小变化时自动触发，实现 “缩放窗口实时更新数值”；
- `moveEvent`：窗口位置变化时自动触发，实现 “拖动窗口实时更新数值”；



## 二、QPainter 绘图框架设计

Qt 绘图的核心是 `QPainter` 类，搭配自定义控件重写 `paintEvent()` 函数，就能实现各种图形绘制。<img src="C:\Users\张超然\AppData\Roaming\Typora\typora-user-images\image-20260128222151590.png" alt="image-20260128222151590" style="zoom:50%;" />

1. **核心类分工**

   - `PaintArea` 类：自定义 Widget，重写 `paintEvent()` 函数，负责具体图形的显示
   - `MainWidget` 类：主窗口类，负责管理图形参数的选择，比如形状、颜色等

2. **常用绘制函数**

   Qt 提供了丰富的 `draw` 系列函数，满足基础图形绘制需求：

   - 绘制直线：`drawLine()`
   - 绘制矩形：`drawRect()`
   - 绘制圆角矩形：`drawRoundedRect()`
   - 绘制椭圆：`drawEllipse()`
   - 绘制多边形：`drawPolygon()`
   - 绘制圆弧：`drawArc()`
   - 绘制路径：`drawPath()`
   - 绘制文字：`drawText()`

3. **代码案例**

   在这里我们先实现一个基于 Qt 框架开发的一个自定义绘图控件类 `PainterArea`，核心作用是**封装可配置的基本图形绘制功能**

   **头文件 `painterarea.h`**

   ```cpp
   #ifndef PAINTERAREA_H    // 头文件保护，防止重复包含
   #define PAINTERAREA_H
   
   #include <QWidget>       // 引入Qt基础窗口控件头文件
   #include <QPainter>      // 引入Qt绘图核心类头文件
   
   // 自定义绘图控件类，继承自QWidget，具备窗口/控件基础特性
   class PainterArea : public QWidget
   {
       Q_OBJECT    // Qt的元对象宏，启用信号槽、属性系统等核心功能
   
   public:
       // 枚举类型：定义支持绘制的图形类型
       enum Shape {
           Line,           // 直线
           Rectangle       // 矩形（可扩展：椭圆、多边形等）
       };
   
       // 构造函数：parent指定父控件，用于Qt的父子对象内存管理
       explicit PainterArea(QWidget *parent = nullptr);
   
       // ************************ 外部配置接口 ************************
       // 设置要绘制的图形类型
       void setShape(Shape shape);
       // 设置画笔样式（颜色、宽度、线型等）
       void setPen(const QPen &pen);
       // 设置画刷样式（填充颜色、纹理等）
       void setBrush(const QBrush &brush);
       // 设置填充规则（针对复杂图形的填充逻辑）
       void setfillrule(Qt::FillRule rule);
   
   protected:
       // 重写Qt的重绘事件函数（核心）：控件需要绘制/更新时自动调用
       void paintEvent(QPaintEvent *event) override;
   
   private:
       // 私有成员变量：存储绘图配置（封装性）
       Shape shp;              // 当前要绘制的图形类型
       QPen pen;               // 画笔对象（控制线条样式）
       QBrush brsh;            // 画刷对象（控制填充样式）
       Qt::FillRule fillrle;   // 填充规则（如Qt::WindingFill/Qt::OddEvenFill）
   };
   
   #endif // PAINTERAREA_H
   ```

   **代码 `painterarea.cpp`**

   ```cpp
   #include "painterarea.h"   // 引入对应的头文件
   
   // 构造函数实现：初始化控件基础属性
   PainterArea::PainterArea(QWidget *parent)
       : QWidget(parent)      // 调用父类QWidget的构造函数
   {
       // 设置控件背景色为红色（调试/区分区域用，可根据需求修改）
       this->setStyleSheet("background-color: red;");
       // 开启背景自动填充：避免背景透明导致绘图闪烁
       this->setAutoFillBackground(true);
       // 设置控件最小尺寸：保证绘图区域足够显示图形
       this->setMinimumSize(410, 410);
   }
   
   // 设置绘制图形类型：外部调用此函数指定要画直线/矩形
   void PainterArea::setShape(Shape shape)
   {
       shp = shape;    // 更新私有成员变量
       update();       // 触发控件重绘：调用paintEvent刷新显示
   }
   
   // 设置画笔样式：外部传入QPen对象（包含颜色、宽度、线型等）
   void PainterArea::setPen(const QPen &pen)
   {
       this->pen = pen;// 更新画笔配置
       update();       // 触发重绘，使新画笔样式生效
   }
   
   // 设置画刷样式：外部传入QBrush对象（控制图形填充）
   void PainterArea::setBrush(const QBrush &brush)
   {
       brsh = brush;   // 更新画刷配置
       update();       // 触发重绘
   }
   
   // 设置填充规则：针对闭合图形的填充逻辑
   void PainterArea::setfillrule(Qt::FillRule rule)
   {
       fillrle = rule; // 更新填充规则
       update();       // 触发重绘
   }
   
   // 核心绘图逻辑：重写paintEvent，实现自定义绘制
   void PainterArea::paintEvent(QPaintEvent *event)
   {
       // 1. 创建QPainter对象，关联当前控件作为绘图画布
       QPainter painter(this);
       // 2. 应用画笔配置：后续绘制的线条使用此画笔
       painter.setPen(pen);
       // 3. 应用画刷配置：后续闭合图形的填充使用此画刷
       painter.setBrush(brsh);
   
       // 4. 定义绘图区域：固定坐标(55,110)，宽290，高180
       QRect rect(55, 110, 290, 180);
   
       // 5. 根据当前图形类型，绘制对应图形
       switch (shp) {
       case Line: // 绘制直线：矩形的对角线（左上到右下）
           painter.drawLine(rect.topLeft(), rect.bottomRight());
           break;
       case Rectangle: // 绘制矩形：使用定义的rect区域
           painter.drawRect(rect);
           break;
       default: // 默认无操作：可扩展其他图形
           break;
       }
   
       // 注：填充规则fillrle暂未实际使用，可扩展如drawPolygon时添加：
       // painter.setFillRule(fillrle);
   }
   ```

## 三、双缓冲机制：解决绘图闪烁问题

这是 Qt 绘图中优化体验的关键技术，能有效避免绘图时的屏幕闪烁。<img src="C:\Users\张超然\AppData\Roaming\Typora\typora-user-images\image-20260129203343789.png" alt="image-20260129203343789" style="zoom:67%;" />

1. **双缓冲机制原理**

   双缓冲不是直接在控件上绘图，而是分两步执行：

   - 第一步：先将所有需要绘制的内容绘制到一个**内存中的图片对象**里
   - 第二步：将这个图片一次性绘制到目标控件上

2. **实战应用场景**

   以绘图工具为例，一般会这样搭建结构：

   - 主窗口：使用 `QMainWindow` 作为程序主界面
   - 工具栏：`QToolBar` 提供线型、线宽、颜色等选择功能
   - 绘图区：自定义 `QWidget` 作为中央窗体，重写 `mousePressEvent`、`mouseMoveEvent`、`paintEvent` 三个事件函数，分别处理鼠标按下、拖动和绘图逻辑

3. **代码案例**

   首先实现一个绘图的子类方便直接进行绘图程序

   **头文件`drawwidget.h`**

   ```cpp
   #ifndef DRAWWIDGET_H
   #define DRAWWIDGET_H
   
   #include <QWidget>
   #include <QtGui>
   #include <QMouseEvent> // 鼠标事件
   #include <QPaintEvent> // 绘制事件
   #include <QResizeEvent> // 实时获取窗口操作等大小
   #include <QColor>
   #include <QPixmap> // 显示图像
   #include <QPainter>
   #include <QPalette>
   #include <QPen>
   
   
   class DrawWidget : public QWidget
   {
       Q_OBJECT
   public:
       / 构造函数：
       // explicit：防止隐式类型转换（Qt类构造函数的最佳实践）
       // parent = nullptr：父窗口指针，默认无父窗口（Qt父子对象机制自动管理内存）
       explicit DrawWidget(QWidget *parent = nullptr);
   
       // ========== 重写父类QWidget的事件函数（响应窗口/鼠标事件） ==========
       // 鼠标按下事件：鼠标点击控件时触发，用于记录绘图起点
   ```

   ```cpp
   DrawWidget::DrawWidget(QWidget *parent) : QWidget(parent)

   {
       // 1. 创建新的QPixmap，大小和当前控件一致
       QPixmap *cPix = new QPixmap(size());
       // 2. 将新画布填充为白色（空白）

       // 3. 替换原有画布为新的空白画布
       pix = cPix;
       // 4. 触发paintEvent重绘：控件显示新的空白画布
       update();
   }

   // 参数e：鼠标事件对象，包含点击位置、按键状态等信息
   void DrawWidget::mousePressEvent(QMouseEvent *e)
   {
       // 保存鼠标按下时的坐标（相对于当前控件的位置）
       startpos = e->pos();
   }
   
   // 重写鼠标移动事件：核心绘图逻辑，鼠标拖动时绘制线条
   // 参数e：鼠标事件对象，包含当前鼠标位置
   void DrawWidget::mouseMoveEvent(QMouseEvent *e)
   {
       // 1. 创建QPainter对象：用于绘制线条（Qt绘图必须通过QPainter）
       QPainter *painter = new QPainter;
       // 2. 创建QPen对象：设置画笔样式、宽度、颜色
       QPen pen;
       // 设置画笔样式：将int类型的style转为Qt::PenStyle枚举（如实线/虚线）
       pen.setStyle((Qt::PenStyle)style);
       // 设置画笔宽度：对应setWidth设置的值
       pen.setWidth(widthss);
       // 设置画笔颜色：对应setColor设置的值
       pen.setColor(color);
   
       // 3. 开始在pix画布上绘制（离线绘制，不直接显示到控件）
       painter->begin(pix);
       // 4. 给画家设置画笔
       painter->setPen(pen);
       // 5. 绘制直线：从上次的起始点到当前鼠标位置
       painter->drawLine(startpos, e->pos());
       // 6. 结束绘制：释放绘图资源
       painter->end();
   
       // 7. 更新起始点：将当前鼠标位置设为下一次绘制的起点（实现连续线条）
       startpos = e->pos();
       // 8. 触发paintEvent重绘：将pix上的绘制内容显示到控件
       update();
   }
   
   // 重写绘制事件：将离线画布pix的内容显示到控件上
   // Qt中所有控件的绘制都必须在paintEvent中完成
   void DrawWidget::paintEvent(QPaintEvent *)
   {
       // 1. 创建QPainter对象，指定绘制目标为当前控件（this）
       QPainter painter(this);
       // 2. 将pix画布绘制到控件的(0,0)位置（左上角）
       painter.drawPixmap(QPoint(0,0), *pix);
   }
   
   // 重写窗口大小变化事件：自适应缩放画布，避免窗口放大后画布空白
   // 参数event：大小变化事件对象，包含旧尺寸/新尺寸
   void DrawWidget::resizeEvent(QResizeEvent *event)
   {
       // 判断：如果控件新尺寸超过原有画布尺寸（宽或高变大）
       if(height() > pix->height() || width() > pix->width())
       {
           // 1. 创建新的更大的画布，尺寸和当前控件一致
           QPixmap *newPix = new QPixmap(size());
           // 2. 新画布填充为白色
           newPix->fill(Qt::white);
   
           // 3. 创建QPainter，将原有画布内容绘制到新画布上
           QPainter ps(newPix);
           // 把旧画布的内容画到新画布的左上角，保留原有绘图内容
           ps.drawPixmap(QPoint(0,0), *pix);
   
           // 4. 替换为新画布：后续绘图在新画布上进行
           pix = newPix;
       }
   
       // 调用父类的resizeEvent：执行Qt默认的大小变化处理（必须加，避免父类逻辑丢失）
       QWidget::resizeEvent(event);
   }
   ```
